# Project Blueprint

## Overview

This project is a modern, responsive e-commerce website for an Ayurvedic products brand called Sahyadri Ayurveda. It features a clean, premium design, a product catalog, a shopping cart, and an AI-powered chatbot for customer assistance.

## Implemented Features

### Design and Layout

*   **Branding:** A professional logo is used in the header.
*   **Typography:** A combination of `Cinzel Decorative`, `Montserrat`, and `Playfair Display` fonts are used for a sophisticated and readable text hierarchy.
*   **Color Palette:** The color scheme uses a mix of natural greens, warm golds, and clean whites to create a premium and calming feel.
*   **Responsive Design:** The layout is fully responsive and adapts to different screen sizes, ensuring a seamless experience on both mobile and desktop devices.
*   **Interactive Elements:** The site includes interactive elements like buttons with hover effects, a shopping cart, and a chatbot.

### Pages and Navigation

*   **Header:** A consistent header with a logo and navigation links is present on all pages.
*   **Footer:** A consistent footer is present on all pages.
*   **Home Page (`index.html`):** The main landing page, featuring a hero section, best-selling products, and sections for "Our Story" and "Why Choose Us?".
*   **Shop Page (`shop.html`):** A page that displays all available products.
*   **Our Story Page (`our-story.html`):** A page that provides more information about the brand.
*   **Consultation Page (`consultation.html`):** A page for users to book a consultation.
*   **Contact Page (`contact.html`):** A page with contact information.

### E-commerce Functionality

*   **Product Display:** Products are displayed in a grid with images, descriptions, prices, and Sanskrit shlokas with translations.
*   **Shopping Cart:** A slide-in shopping cart allows users to add, remove, and change the quantity of items. The cart total is updated automatically.
*   **Add to Cart:** Products can be added to the cart from both the home and shop pages.

### AI Chatbot

*   **Vedic Healer AI:** An AI-powered chatbot is available to answer customer questions about the products and their uses.
*   **Keyword-Based Responses:** The chatbot provides responses based on keywords in the user's query.
